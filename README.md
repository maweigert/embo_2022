### Setup 

1. Clone this repo:

  `git clone git@github.com:maweigert/embo_2022.git`

2. Install the conda environment 

  `conda env create -f environment.yaml`


3. Activate environment and start jupyter

  ```
  conda activate stardist
  jupyter notebook
  ```
